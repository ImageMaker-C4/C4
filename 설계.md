# C4 — AI Image Lab 설계 문서

## 프로젝트 개요

텍스트 프롬프트를 입력하면 AI가 이미지를 생성해주는 채팅 기반 웹 앱입니다.
백엔드 서버 없이 브라우저에서 직접 AI API를 호출하는 구조로 설계되어 있습니다.

---

## 아키텍처

```
사용자 브라우저
    │
    ▼
React (Vite) — 정적 프론트엔드
    │
    ▼ HTTPS API 호출
fal.ai — SDXL-Lightning 이미지 생성
```

**서버가 없습니다.** 프론트엔드가 fal.ai API를 직접 호출합니다.
DB, 백엔드, 인프라 비용이 전혀 없습니다.

---

## 기술 스택

| 항목 | 기술 |
|------|------|
| 프론트엔드 | React 18 + TypeScript |
| 번들러 | Vite |
| 스타일 | CSS (커스텀 디자인 시스템) |
| 이미지 생성 AI | fal.ai — SDXL-Lightning 4step |
| 백엔드 | **없음** |
| DB | **없음** |

---

## 이미지 생성 API 선정 이유 (가성비 분석)

### 선택: [fal.ai](https://fal.ai) — `fast-lightning-sdxl`

| 항목 | 내용 |
|------|------|
| 모델 | SDXL-Lightning 4step (ByteDance) |
| 생성 속도 | 약 1~3초 |
| 비용 | 이미지 1장당 약 $0.00055 (1,000장 = $0.55) |
| 무료 크레딧 | 가입 시 $10 제공 (약 18,000장 생성 가능) |
| 한국어 지원 | 가능 |
| 서버 필요 여부 | 없음 (브라우저 직접 호출) |

### 비교 검토

| 서비스 | 속도 | 비용/장 | 서버 필요 | 비고 |
|--------|------|---------|-----------|------|
| **fal.ai SDXL-Lightning** | 1~3초 | $0.00055 | ❌ | **현재 선택** |
| Replicate SDXL-Lightning | 3~8초 | $0.0023 | ❌ | 비동기 폴링 필요 |
| Stable Diffusion (로컬) | 5~30초 | 전기세만 | ❌ | GPU 필요 |
| DALL-E 3 (OpenAI) | 10~20초 | $0.04 | ❌ | 73배 비쌈 |
| Midjourney | 빠름 | 월 $10~ | ❌ | API 없음 |

→ fal.ai가 **속도, 비용, 구현 단순성** 세 가지 모두 우수합니다.

## 로컬 실행 방법 (서버 없이 바로 실행)

> 별도 서버 없이 누구나 로컬에서 실행 가능합니다.

### 요구 사항
- Node.js 18+
- fal.ai API 키 ([발급 링크](https://fal.ai/dashboard/keys))

### 실행

```bash
# 1. 레포 클론
git clone https://github.com/ImageMaker-C4/C4.git
cd C4

# 2. 의존성 설치
npm install

# 3. 환경변수 설정
echo "VITE_FAL_KEY=여기에_발급받은_키_입력" > .env

# 4. 실행
npm run dev
```

브라우저에서 `http://localhost:5173` 접속하면 바로 사용 가능합니다.

### 정적 빌드 (배포용)

```bash
npm run build
# dist/ 폴더를 Vercel, Netlify, GitHub Pages 등에 업로드
```
