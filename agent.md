# AI 이미지 생성 웹사이트 (AI Image Generator)

**프로젝트 개요**: 사용자가 입력한 프롬프트(묘사)를 바탕으로 그에 맞는 이미지를 실시간으로 생성하고 보여주는 웹사이트입니다.

## 1. 주요 기능 (Core Features)
- **챗 기반 이미지 생성**: 사용자의 텍스트 입력에 대해 1:1로 대응하는 이미지 생성 및 응답
- **이미지 갤러리**: 생성된 결과물들을 한눈에 확인하고 관리하는 기능

## 2. 팀 구성 및 역할 분담 (Team Roles)

### Agent: Frontend Lead (하동건)
- React/Vite 기반 웹 UI 개발
- 프롬프트 입력창 및 실시간 이미지 렌더링 구현
- 챗 형식의 인터페이스 및 상태 관리

### Agent: Backend Architect (이지훈)
- FastAPI 기반 API 서버 구축 및 관리
- 이미지 생성 엔진(Pollinations.ai) 연동 및 데이터 처리
- 데이터베이스(PostgreSQL) 설계 및 생성 내역 저장

### Agent: AI Specialist (정명우)
- 이미지 생성 모델 파라미터 최적화
- 프롬프트 품질 개선 및 결과물 품질 관리
- (추후) 유해 콘텐츠 필터링 로직 검토
